我的学校

返回数据里有schoolInfo:学校信息,applyInfo:申请入校信息，两个不会同时有

POST/classService/api/school/getMySchoolInfo
请求 Request

QUERY

  • tokenstring必填

    用户token

响应 Response
状态码 200

参数

  • resultnumber

    0:成功，!0:失败

  • msgstring

    提示消息

  • dataobject

    返回数据

    • 子属性
    • data.schoolInfoobject必填

      学校

    • data.applyInfoobject必填

文本
POST
{{BASE_URL}}/classService/api/school/getMySchoolInfo?token=fardarfdasresaf%3D%3Dartfesa

1HTTP/1.1 200 OK
2{
3  result: 0,
4  msg: "SUCCESS",
5  data: {schoolInfo:{schoolId: 1, name: "第一小学", province: 350000, city: 350400, county: 350425}, applyInfo: {schoolId: 1,schoolName: "第一中学", status: 0}}

注册学校

根据省，市，区县，还有学段来注册学校，省和学段必填，市，区县选填

POST/classService/api/school/registSchoolInfo
请求 Request

QUERY

  • areaidnumber必填

    地域id

  • phaseTypenumber必填

    学段id

  • pidnumber必填

    上级学校id

  • modestring必填

    学段组合{Junior: 3}

  • schoolNamestring必填

    学校名字

  • commentstring必填

    学校全称

  • sameNamestring必填

    同地区是否允许同名学校，0不允许，1允许

  • modeAdminnumber必填

    加入模式：1禁止加入 2自由加入 4批准加入

  • propertynumber必填

    学校属性 1省直属，2市直属，0无

  • idCodestring必填

    学校识别码

  • contactNamestring必填

    联系人

  • contactPhonestring必填

    联系电话

  • tokenstring必填

    用户token

响应 Response
状态码 200

参数

  • resultnumber

    0:成功，!0:失败

  • msgstring

    提示消息

  • dataobject

    返回数据

    • 子属性
    • data.schoolIdnumber必填

      学校id

    • data.namestring必填

      学校名字

Example 1

Request-Example:

文本
POST
{{BASE_URL}}/classService/api/school/registSchoolInfo?token=fardarfdasresaf%3D%3Dartfesa&phaseType=1&pid=0&mode=%7BJunior%3A3%7D&schoolName=%E7%AC%AC%E4%B8%80%E4%B8%AD%E5%AD%A6&comment=%E6%B9%96%E9%87%8C%E5%8C%BA%E7%AC%AC%E4%B8%80%E4%B8%AD%E5%AD%A6&sameName=0&modeAdmin=4&idCode=3545124&contactName=%E5%BC%A0%E4%B8%89&contactPhone=137%2A%2A%2A%2A%2A%2A&areaid=350425&property=0

2{
3  "result": 0,

老师获取全校职工的手机号和自己班级家长的手机号，如果没有手机号就不显示。如果老师有多个班级，要显示所有班级名字

POST/classService/api/phone/getSchoolPhones
请求 Request

QUERY

  • tokenstring必填

    用户token

响应 Response
状态码 200

参数

  • resultnumber

    0:成功，!0:失败

  • msgstring

    提示消息

    • data.rolenumber必填
